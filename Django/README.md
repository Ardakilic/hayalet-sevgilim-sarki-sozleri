### Create virtual env for library

| OS        | Create virtualenv | Activate virtualenv      |
| --------- | ----------------- | ------------------------ |
| windows   | py -m venv venv   | .\venv\Scripts\activate  |
| mac/linux | virtualenv venv   | source venv/bin/activate |

### Start project

py manage.py runserver
