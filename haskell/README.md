# Hayalet Sevgilim Sarki Sozleri

## Build

You need `nix` for reproducible build and dev environments.

```sh
nix build
```

## Run
```sh
nix run
```

## Develop

You may need `direnv` and run this when loading the directory.

```sh
direnv allow
```

Your build tools will be loaded. Now you can do this.

```sh
cabal run
```
